# Sales & Revenue Dashboard

## Project Description
This project analyzes sales and revenue data using Python and provides insights through visualization.

## Features
- Total Revenue calculation
- Sales analysis by product
- Daily revenue trend
- Product performance comparison

## Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit (for local dashboard)

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   streamlit run sales.py
